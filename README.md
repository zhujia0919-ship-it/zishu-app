# 字·树 —— 小学生识字 App（演示版）

帮助 1-6 年级小朋友识字：**甲骨文追溯字源 → 动画演示演变 → 吕佛庭文字画 → 组词朗读**。

基于 React 18 + Vite，移动优先。目前收录一年级象形字 20 个，按「大自然 / 我的身体 / 小动物」分组：

- 大自然：木 日 月 山 水 火 石 田 禾 草 云 雨
- 我的身体：人 口 目 耳 手 大
- 小动物：鸟 鱼

选字以部编版一年级教材的象形字为主（如《日月水火》《口耳目》等识字课）。每个字的甲骨文字形为本项目手绘 SVG（甲骨文为三千年前古文字，字形本身无版权问题）。「木」配有真实演变动画视频；其余字自动显示内置的「甲骨文 ↔ 楷书」渐变演变动画，待正式动画就位后填入路径即可替换。

## 功能

- **首页字卡**：大字 + 拼音 + 甲骨文角标，每个字一种主题色
- **字详情页**（学习闭环）：
  1. 三千年前（甲骨文）➜ 今天（楷书）对照，点 🔊 朗读字音
  2. 「它像什么」一句话联想
  3. 「字的来历」字源小故事
  4. 演变动画（甲骨文 → 金文 → 小篆 → 隶书 → 楷书）
  5. 吕佛庭文字画欣赏
  6. 组词卡片，点一下即朗读
  7. 拓展小知识：文字画三种构图
- **语音朗读**：使用浏览器内置中文语音合成（Web Speech API），免费、离线可用、无需任何接口

## 本地运行

需要 Node.js 18+。

```bash
npm install
npm run dev
```

## 部署

与静态站点相同：推到 GitHub → Vercel Import → Deploy（详细步骤同文人生活美学堂项目）。

## 如何添加新字

打开 `src/data/chars.js`，复制一段 `{ ... }` 字对象，修改：

- `char / pinyin / strokes`：字、拼音、笔画
- `oracle`：甲骨文写法（可先用近似字符，之后换图片）
- `like / story`：像什么、字源故事
- `anim`：演变动画视频，放进 `public/assets/` 后填路径；没有填 `null`
- `painting / paintingNote`：文字画图片与说明；没有填 `null`
- `words`：组词
- `color`：卡片主题色
